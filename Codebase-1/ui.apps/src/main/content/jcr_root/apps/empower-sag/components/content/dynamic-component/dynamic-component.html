Description

PIE-71643

A default value specified for a field in a gRPC request is not passed to the input pipeline of the gRPC method flow service.
In gRPC, fields have type-specific default values. When a gRPC request included one or more fields set to the default for the field type, Integration Server did not include the value in the data passed to the input pipeline of the gRPC method flow service.
This issue is now resolved.
